# Camera Calibration Addon для Blender

Аддон для калибровки камер на основе изображений, позволяющий точно восстанавливать положение камер в 3D пространстве.

## Требования

- Blender 3.0+
- Python 3.7+
- Зависимости: numpy, scipy, matplotlib, opencv-python

## Установка

1. Скачайте архив с аддоном
2. В Blender откройте Edit > Preferences > Add-ons > Install
3. Выберите скачанный архив и нажмите "Install Add-on"
4. Активируйте аддон "Camera Calibration"
5. После активации аддона вы увидите предупреждение об отсутствующих зависимостях

## Установка зависимостей

### Автоматическая установка через UI

1. После активации аддона откройте его настройки в Preferences > Add-ons
2. Вы увидите раздел "Camera Calibration Dependencies" с списком необходимых зависимостей
3. Нажмите кнопку "Установить зависимости"
4. Дождитесь завершения установки и перезапустите Blender

### Установка через консоль Blender

1. Откройте Blender и активируйте аддон
2. Откройте консоль в Blender: Window > Toggle System Console (для Windows) или запустите blender из терминала (для macOS/Linux)
3. В настройках аддона нажмите "Установить зависимости" и следите за прогрессом в консоли
4. Перезапустите Blender после установки

### Ручная установка зависимостей

Если автоматическая установка не работает, вы можете установить зависимости вручную:

1. Откройте командную строку/терминал от имени администратора
2. Перейдите в директорию с Python, который использует Blender:
   - Windows: `cd "C:\Program Files\Blender Foundation\Blender X.X\X.X\python\bin"`
   - macOS: `cd /Applications/Blender.app/Contents/Resources/X.X/python/bin`
   - Linux: `cd /path/to/blender/X.X/python/bin`
3. Выполните команду:
   ```
   python -m pip install numpy scipy matplotlib opencv-python
   ```

## Проверка зависимостей

Чтобы убедиться, что все зависимости установлены правильно:

1. В Preferences > Add-ons найдите Camera Calibration
2. Откройте раздел "Camera Calibration Dependencies"
3. Убедитесь, что все зависимости имеют статус "Установлена"
4. Если необходимо, нажмите кнопку "Проверить зависимости" для повторной проверки

## Использование

1. Откройте боковую панель в режиме 3D View (клавиша N)
2. Выберите вкладку "Image Matching"
3. Загрузите изображения, нажав на кнопку со значком папки
4. Создайте группы точек (каждая группа представляет одну точку в 3D пространстве)
5. Откройте редактор изображений и разместите точки на каждом изображении
6. Запустите калибровку, нажав кнопку "Расширенная калибровка"
7. После успешной калибровки можно:
   - Создать камеры в сцене Blender
   - Сохранить результаты калибровки в файл
   - Визуализировать результаты

## Решение проблем

### Ошибка "ModuleNotFoundError: No module named 'scipy'"

Эта ошибка означает, что не установлена библиотека SciPy. Возможные решения:

1. Воспользуйтесь автоматической установкой зависимостей в настройках аддона
2. Установите зависимости вручную, как описано выше
3. В некоторых случаях может помочь запуск Blender от имени администратора

### Аддон работает не полностью

Если аддон загружается, но функции калибровки недоступны:

1. Проверьте статус зависимостей в настройках аддона
2. Убедитесь, что вы перезапустили Blender после установки зависимостей
3. Проверьте консоль Blender на наличие сообщений об ошибках

## Поддержка

Если у вас возникли проблемы с установкой или использованием аддона:

1. Проверьте наличие всех необходимых зависимостей
2. Убедитесь, что на изображениях отмечено достаточное количество общих точек (минимум 8 точек на пару изображений)
3. Качество калибровки зависит от количества и распределения точек на изображениях

## Особенности

- Высокоточная калибровка камер
- Поддержка различных форматов изображений
- Оптимизация с помощью Bundle Adjustment
- Визуализация результатов
- Сохранение и загрузка данных калибровки
- Автоматическое создание камер в сцене Blender

## Разработка

Аддон имеет модульную структуру для упрощения поддержки и расширения. Основные компоненты:

- `calibration_modules/` - модули для калибровки камер
- `calibration.py` - основной интерфейс калибровки
- `calibration_bridge.py` - мост между UI аддона и модулями калибровки
- `image_editor.py` - редактор для размещения точек на изображениях
- `dependencies.py` - модуль для управления зависимостями
