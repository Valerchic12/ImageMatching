# Модуль кривых Безье для Blender

Модуль для автоматического создания точек калибровки с помощью кривых Безье в редакторе изображений.

## Описание

Этот модуль добавляет функциональность кривых Безье, которая позволяет создавать множество точек калибровки, 
расположенных по кривой между двумя заданными точками. Вы можете:

- Создавать кривые Безье, устанавливая начальную и конечную точки
- Редактировать форму кривой, перемещая контрольные точки
- Настраивать количество контрольных точек для более точного управления формой кривой
- Настраивать количество точек калибровки, генерируемых на кривой
- Предварительно просматривать точки калибровки перед их добавлением

## Структура модуля

Модуль состоит из следующих файлов:

- `__init__.py` - Инициализация модуля, экспорт необходимых функций
- `bezier_curves.py` - Основная логика работы с кривыми Безье
- `bezier_ui.py` - Пользовательский интерфейс и обработка ввода
- `integration_example.py` - Примеры интеграции с основным кодом
- `README.md` - Документация

## Интеграция с существующим аддоном

Для интеграции модуля с существующим аддоном необходимо выполнить следующие шаги:

### 1. Установка модуля

Разместите папку `bezier_module` в директории вашего аддона.

### 2. Импорт необходимых функций

В соответствующих файлах вашего аддона добавьте импорт нужных функций:

```python
# В файле image_editor.py
from bezier_module.bezier_ui import (
    initialize_bezier_manager,
    handle_bezier_mode_input,
    draw_bezier_overlay,
    update_bezier_hover_state
)
```

### 3. Инициализация менеджера кривых

В методе `invoke()` оператора редактора изображений добавьте инициализацию менеджера:

```python
def invoke(self, context, event):
    # ... существующий код ...
    
    # Инициализируем менеджер кривых Безье
    initialize_bezier_manager()
    
    # ... существующий код ...
    return {'RUNNING_MODAL'}
```

### 4. Обработка ввода

В методе `modal()` оператора редактора изображений добавьте обработку ввода для кривых Безье:

```python
def modal(self, context, event):
    # ... существующий код ...
    
    # Получаем текущее изображение и позицию мыши
    image_item = self.get_current_image_item()
    if image_item:
        # Преобразование координат мыши в координаты изображения
        mouse_pos_region = (event.mouse_region_x, event.mouse_region_y)
        mouse_pos_image = self.region_to_image_coords(context, image_item, 
                                                     mouse_pos_region[0], 
                                                     mouse_pos_region[1])
        
        # Обновляем состояние наведения для контрольных точек кривой Безье
        update_bezier_hover_state(self, context, mouse_pos_image)
        
        # Проверяем активен ли режим кривой Безье
        if context.scene.bezier_props.placement_mode == 'BEZIER_CURVE':
            # Обработка ввода для кривых Безье
            if handle_bezier_mode_input(self, context, event, image_item, mouse_pos_image):
                # Если событие обработано в режиме кривой Безье, пропускаем обычную обработку
                return {'RUNNING_MODAL'}
    
    # ... существующий код ...
    return {'PASS_THROUGH'}  # или другой результат в зависимости от обработки события
```

### 5. Отрисовка кривых

В методе `draw_callback()` оператора редактора изображений добавьте отрисовку кривых Безье:

```python
def draw_callback(self, context):
    # ... существующий код ...
    
    # Получаем текущее изображение

        draw_bezier_overlay(self, context, shader, image_item, params)
    
    # ... существующий код ...
```

### 7. Пользовательский интерфейс

Добавьте элементы управления кривыми Безье в панель инструментов вашего аддона:

```python
def draw(self, context):
    layout = self.layout
    scene = context.scene
    
    # ... существующий код ...
    
    # Добавляем управление режимом кривой Безье
    box = layout.box()
    box.label(text="Режим расстановки точек:")
    box.prop(scene.bezier_props, "placement_mode", expand=True)
    
    # Если выбран режим кривой Безье, показываем дополнительные настройки
    if scene.bezier_props.placement_mode == 'BEZIER_CURVE':
        # Настройки кривой
        box = layout.box()
        box.label(text="Настройки кривой Безье:")
        box.prop(scene.bezier_props, "num_control_points")
        box.prop(scene.bezier_props, "num_curve_points")
        box.prop(scene.bezier_props, "show_preview")
        
        # Кнопки действий
        row = layout.row()
        row.operator("camera_calibration.apply_bezier_points")
        row.operator("camera_calibration.clear_bezier")
    
    # ... существующий код ...
```

### 8. Добавление точек калибровки

В операторе `CAMCALIB_OT_apply_bezier_points` добавьте код для интеграции с вашей системой точек:

```python
def execute(self, context):
    # ... существующий код ...
    
    # Получаем точки с кривой
    from bezier_module import get_bezier_points
    points = get_bezier_points(context)
    
    # Добавляем точки в систему калибровки
    props = context.scene.camcalib
    props.add_points_to_images(context, points)
    
    # ... существующий код ...
    return {'FINISHED'}
```

## Использование в интерфейсе

После интеграции модуля в аддон пользователь может:

1. Выбрать режим "Кривая Безье" в панели инструментов
2. Щелкнуть в редакторе изображений для установки начальной точки
3. Щелкнуть в другом месте для установки конечной точки
4. Настроить параметры кривой (количество контрольных точек, количество точек калибровки)
5. При необходимости отредактировать форму кривой, перетаскивая контрольные точки
6. Нажать кнопку "Применить точки" для добавления точек калибровки

## Примечания по совместимости

- Модуль тестировался на Blender версии 3.0.0 и выше
- Для корректной работы требуется совместимый редактор изображений с доступом к преобразованию координат
- Если ваш аддон использует свою систему отрисовки, может потребоваться адаптация функций отрисовки
